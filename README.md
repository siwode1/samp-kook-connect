此插件允许你在游戏模式中控制 Kook 机器人。

### 安装
你需要先创建一个机器人，以便将机器人令牌传递给插件。

1. 访问 https://developer.kookapp.cn/
2. 创建一个新的应用/机器人
3. 输入机器人名称
4. 创建应用
5. 从机器人设置中获取你的机器人令牌

现在你有了机器人令牌！

要将机器人添加到你的服务器，你需要通过 Kook 开发者门户或邀请链接使用机器人的客户端 ID 邀请它。

现在只需在 server.cfg 文件中添加这一行就可以了！

`kook_bot_token YOUR_BOT_TOKEN_HERE`

或者，你可以使用环境变量 `SAMP_KOOK_BOT_TOKEN` 设置机器人令牌。

## 注意事项
* 永远不要与任何人分享你的机器人令牌
* 确保你的服务器时钟定期同步，否则速率限制器将无法正常工作，你的机器人可能会被封禁
* 此插件专为 open.mp (OMP) 服务器设计

## OMP 配置
对于 open.mp 服务器，你也可以在服务器配置文件中配置机器人令牌：

```
kook.bot_token YOUR_BOT_TOKEN_HERE
```
